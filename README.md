# 📈 Stock Market Analysis & Prediction System

A Machine Learning-powered **Stock Market Analysis & Prediction System** built with **Python**, **Streamlit**, **Scikit-Learn**, **Yahoo Finance API**, and **Plotly**.
This application provides real-time stock analysis, technical indicators, AI-based recommendations, interactive charts, and next-day stock price prediction for major Indian stocks and market indices.

# ✨ Features

## 📊 Live Stock Data
* Real-time stock prices using Yahoo Finance
* Live market status (Open/Closed)
* Automatic data refresh every 60 seconds
* Latest price updates
  
## 🤖 Machine Learning Prediction
The trained Random Forest model predicts:
* Tomorrow's Closing Price
* Tomorrow's Opening Price (Estimated)
* Expected Percentage Change
* Bullish/Bearish Trend

## 💡 AI Recommendation System
The application generates intelligent recommendations based on:
* Machine Learning Prediction
* RSI
* MACD
* Moving Average Crossover

Possible recommendations:
* 🟢 STRONG BUY
* 🔵 BUY
* 🟡 HOLD
* 🔴 SELL

# ▶️ Installation

Clone the repository:
git clone https://github.com/tarzsardana-ship-it/Stock-Market-Analysis-Prediction.git

Move into the project folder:
cd Stock-Market-Analysis-Prediction

Install dependencies:
pip install -r requirements.txt

Run the application:
streamlit run app.py
